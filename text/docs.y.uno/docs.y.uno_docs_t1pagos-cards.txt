Integration configuration





Credentials and steps to obtain them:

            In the Admin system (Sandbox and Production), in the left menu, click Tokens. See to your list of Access Tokens. 
          


Dashboard configuration:


Log in to dashboard.y.uno.
Click Connections and then select T1Pagos.
Click Connect.
Introduce Name and Access Token.
Click Connect




Configuration of webhooks in partner:

            The register is made from the Admin system (Sandbox and Production), in the left menu, click Webhooks. More than one endpoint can be included and notifications can be channeled to each new route.
