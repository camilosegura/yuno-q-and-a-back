Testing steps

In order to test making a payment using DaviPlata, you should complete the following steps:

Create Customer
Create Checkout Session
Create One Time Token / Get the token from SDK. Use DAVIPLATA as the payment type to generate the One Time Token. 
Create Payment 

The response body of the Create Payment request will provide a redirect URL (payment_method.payment_method_detail.bank_transfer.redirect_url) to the DaviPlata service. On this page, select a document type and fill in the document number with 1 to continue with the payment.
Integration configuration





Credentials and steps to obtain them:


In Daviplata's dashboard click on Aplicaciones.
Select the name of the Application.
The Consumer Key and Consumer Secret will be displayed under Credenciales.




Dashboard configuration:


Log in to dashboard.y.uno.
Click Connections and then select Daviplata.
Click Connect.
Introduce Name, MerchantID, and Terminal ID.
Click Connect
