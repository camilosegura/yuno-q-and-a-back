ReportgetDownload a ReportgetList All ReportsgetRefund Paymentpost https://api-sandbox.y.uno/v1/payments/{id}/transactions/{transaction_id}/refundThis service allows you to create refunds of an existing payment. If the amount field was filled, it will create a partial refund, otherwise it will create a complete refund.
Creating a new refund will refund a charge that was previously created but not yet refunded. The funds will be refunded to the payment method that was originally charged.
