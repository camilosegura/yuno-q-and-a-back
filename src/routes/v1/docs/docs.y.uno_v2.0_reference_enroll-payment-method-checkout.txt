Payment Methodpost https://api-sandbox.y.uno/v1/customers/sessions/{customer_session}/payment-methodsThis request enrolls a payment method for a customer. With the information provided by Yuno after the customer selects the payment method to enroll, you will be able to save it for future purchases in the payment method object created. Note that this request requires an X-Idempotency-Key.
